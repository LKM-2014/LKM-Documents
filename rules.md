# 理科迷理事会管理规范条例

注:本条例是还未成为正式组织的临时守则，但同样也有规范效益，还请各工作组和专业组遵守。

---

## 第一章 各部、各组关系

1. 由于组织的扩大，所有组织人员必须有流变，除了核心人员，必须新老交替并精简
人数，分层分组组织。 
2. 各部、各组织组长，应当只专注于理科迷成员的建设，不能有其他学习组织职务在身。对于理科迷组织需要保守的秘密，均都不可泄露，如有，总务部将降下处分。
3. 总务组是作为理科迷最核心的决策与管理组，将升为总务部，管理各工作组和专业组，是属于管理与被管理的关系。
4. 分群管理员为群务组，是属于总务组的后备成员，受总务组协调，拥有最低的权限，负责各平台的管理与调动氛围，把控风向。平台可以有 QQ 群组、贴吧、知乎等平台管理，但当，有专一平台的管理的时候，可以分为专务组。
5. 活动策划组升为活动策划部，最核心组为常务策划组，并管其他组员的筹备、策划与执行工作。主要包含的为理科迷各平台以及 QQ 群组活动，以后将会扩展为线下业务，下管理专业委员会。未来以后会组织新闻社与专业委员会一起成立相应工作组，翻译最新前沿科普内容，如负责最新国外的讲课讲座、论文、著作等等，助力知识传播。
6. 专业委员会的成分，主要是各平台当中，有相关专业水平、热爱科普事业为主、参加活动为其次的人员。职能是负责产出科普视频、公开课、文章等等形式，将知识传播于群众。
7. IT 技术支持部，简称为 IT 部，也可以称为网络建设部门，会负责理科迷技术支持，网站的建立与维护，如果以后有可能，会向对外合作开展学习组织交流论坛，或是理科迷属于自己的软件。
8. 新闻社，会负责理科迷的对外交流、公关与宣发工作，相应的职能有美工，做宣传海报、拍摄、做视频剪辑等等，同时后续将负责发展线下，并且进行科普类工作传播，如搬运各科前沿内容到各平台;如果后续会扩大，会负责理科迷出版等方面内容，会成为新闻办。
9. 顾问组为重新组为顾问团，将有本体和非本体，主要包含的职能为技术支持与咨询、法律顾问、对外交流学习组织等等。

## 第二章 总务部与群务组

1. 总务组成员的权利  
   - 对领导理科迷发展有大致决定方向的权利。
   - 可以提议各种当前需要，决定新增加减各工作组，以及各组的相应负责人，以方便开展各相应工作。
   - 可以提议各种现在需要，增加或减少群组数量，以及设置相应群务组人员。
   - 开展宣传工作，所有总务组成员都均可对于目前暂时有的宣传途径视频账号，有所控制，发布各种决定以及相应策划。
   - 可以制定开展相应的活动策划，拥有最终的活动解释权。
   - 对于各个理科迷有管理员的权利，能够发布相应的公告和决定，可以进行成员的信息管理，进行放人和踢人的工作。
   - 目前活动经费均为自掏腰包，后续应当有相应的经费财务有相应的管理与监督权利。
   - 可以提出任何不合理的决策异议，以及更好可更换的决策的权利。
   - 如有换届必要，可以开大会，有总务组长的竞选权。
   - 其他有关总务组长规定，以及可以下放的权力。
2. 总务组成员的义务
   - 需要把控好理科迷大致发展方向，要能够保证长线运营的义务。
   - 管理各工作组和专业组以及各群组时，应当合理，不能耽搁任何有损工作进度的义务。
   - 开展宣传工作应当合理，不做任何有损害理科迷声誉的义务。
   - 开展活动策划需要有符合当前情况的义务。
   - 总务组有任何当前群组发生情况，都需要处理的义务，有相应情况的时候，有相应及时发布公告和决定的义务。
   - 虽然现在没有财务上的相关事宜，但是有对总务组财务情况，进行监督以及管理的义务。
   - 有对于目前各种不合理，或者说是能够改进的地方，及时提出的义务。
   - 有其他总务组组长需要安排的任何事物。
3. 现任总务组长为七月花。
4. 总务组组长权利
   - 最终决策并把握理科迷发展方向的权利。
   - 管理并决定理科迷所有工作组成员罢免以及任命、工作项目开展与否、以及工作群组数量增加或减少的权利。
   - 召开相应工作组大会。
   - 决定其他重大事项。
   - 对于理科迷所有事物有着最终的解释权。
5. 总务组组长义务
   - 管理并且对于理科迷负责的义务。
   - 需要倾听关于总务组和群务的任何想法，以及顾问团建议提出的事宜，并给出相应回应。
   - 当任何寻求合作需要沟通时，有出面进行受理的义务。
   - 其他相应需要处理的义务。
6. 群务组的权利
   - 群务组如果扩大了之后，可以专门负责一部分管理，变成专务。
   - 群务组成员有足够能力，可以被总务组成员提及，担任更多相应事务，工作能力强可被提为总务组成员。
7. 群务组的义务
   - 群务组为最低权限，权限仅限分群的管理，义务是需要对分群负责。
   - 处在该分群里应当活跃，以及调整群内的风气，有任何事物可以及时发公告进行处理，按照群规进行处置。
   - 分享更多关于该领域的最新资讯，当下最热门的前沿和交叉事物，引导话题风向。

## 第三章 活动策划部

1. 活动策划组暂时为新任工作部，暂时还不全面，活动策划部长为千寻。
2. 在活动中，当期活动策划组，是按照档期活动内容调整相应成员，也会从有时间并且有能力的专业委员会中进行挑选，属于轮值当期专业组成员。
3. 如果活动策划部人员工作能力，如在策划，筹备，执行能力强，能负责多轮不同的活动筹备，应当为常务的活动策划组组员。
4. 活动策划部成员/常务权利
   - 可以提出新的有意思的活动策划，也可以主动请缨，或是举荐相应可以负责的成员，或是理科迷群组里的成员。
   - 所以对于活动策划、筹备以及执行时不解，或者说是有异议的地方进行提出。
   - 对于关键核心的活动重心，所有成员都均可参与其中，进行一些内容的制定，统一大会商量活动执行时最后的核心内容。
   - 活动策划部常务人员，可以主动请缨进行活动从开始到结束，一条龙服务的负责以监制，可以名为临时活动的活动组长。
   - 在有必要换届时可开大会，常务组员，有可以竞选策划部部长的竞选权。
5. 活动策划部成员/常务义务
   - 在提出相应的活动策划时，应当考虑目前理科迷情况，以及相后续各种相关策划需要考虑到的一些突发情况，给出较为完善的活动，从策划，筹备以及执行时一条龙方案。
   - 部门常务在负责相应活动的时候，作为档期活动策划组组长，应当尽责尽力，考虑周全。
   - 在活动出差错时，应当及时提出并进行正确的相应纠正，如果有任何拿不准的情况，请立即通知部长，如果联系不上，请找相应代理。
   - 在活动执行时有任何突发情况，应当及时安抚情绪，并给出相应的合理理由和建议。
   - 对于活动策划部，策划的所有一些活动关键的内容，都应该有保密的义务，不做有损任何活动最终执行决策的事情。
   - 不做任何有损坏理科迷声誉的行为。
6. 活动策划组组长权利
   - 最终决策并把握活动策划方向、内容以及时宜。
   - 管理并决定，活动策划组成员以及常务罢免以及任命、工作项目开展与否、以及活动数量增加或减少的权利。
   - 对于该部门的人各种表现有所奖惩。
   - 召开相应的活动策划组会议。
   - 决定其他相应重大事项。
   - 对于档期开展的活动有相应的解释权。
7. 活动策划部部长义务
   - 管理并且对于活动策划组成员、常务，以及活动策划组活动负责的义务。
   - 需要倾听关于活动策划组成员，的任何想法，以及提出的事宜，并给出相应回应。
   - 当活动策划部有相应的基本策划案出来时，应当呈交给总务组群，以及总务组组长，进行最终的决策和受理，以及修改。
   - 当活动执行出现问题的时候，有相应的策划组成员或者常务有所通知，如果在线，则需要第一时间出来受理的义务。
   - 当任何寻求合作需要沟通时，应当及时通知总务组，按照相应通知回应进行处理。
   - 对于活动所有的策划，筹备以及执行都应该，在活动结束时进行相应的总结，以及询问参加活动人员相关建议，呈交相应活动总结给总务组。
   - 对于所有实际活动，都应该有相应的文件夹进行保存各种文件的义务，以便下次翻阅。
   - 对于所有的活动策划成员，策划活动应当负责以及监管的义务，及时的做出奖励和惩罚。
   - 对所有活动策划成员，以及专业委员会成员，有相应进行组织的义务，并且对于输出的内容和活动质量进行把关和考核。
   - 有总务组成员或者说是总务组长有安排的工作，有相应执行的各种义务。
