# 理科迷理事会的公约

全体应认同条约，并各工作组应当按管理规范守则做事  

---
1. 理科迷七月团队理事会现有规划为:努力促进和打造，良好可以交流相关理工科的环境
为目的，不断提高团队影响力，成为国家能认证在册的社会组织。  
理科迷团队是专精于理工科发展的组织，现有的宗旨为，让更多人产生上继续保持对理工
科的喜欢和爱，宣传热爱与能力高低和年龄大小无关，让知识流动和得到传播，打破知识
壁垒。未来也将持续的推动科普事业发展，并努力提供更好的科普教育平台，塑造更好的
学术环境与氛围为目标。 
2. 团队所有里的理事会成员，应当提倡，热爱与能力高低和年龄大小无关。  
3. 无论与什么样子的人对话，态度都要有不偏不倚，不歧视不攀比的带头作风，要平等和
尊重的与人交流。  
4. 目前团队晋升暂为，能者得之，应当有一定的能力以及公信力，并且关键的是，要有把
理科迷建设好的决心。   
5. 应当遵守管理规范守则，要明白权利和义务的统一，主动承担。  
6. 不应称自己为领导，高高在上，不可在外炫耀自己理科迷团队任职。  
7. 理科迷任何工作组都只是分工不同，不代表着谁高谁低，谁就应该听谁的，有任何东西
应当探讨。  
8. 对外一致，对内协商。  
9. 成员应当互相信任，互相谅解和包容，有事说事，但不可用情绪做事，不可恶语相向、
阴阳伤害团队共事友谊。  
10. 在外，不做有损理科迷组织名誉的事，涉及到保密的时候，应当守口如瓶。
11. 有任何特殊事情，自己或群成员有如建议、质疑、紧急情况，如判断自身处理不了，
上报给各组长，并在总务群里面吱声，切记不可闷声不响。大情况一起讨论，一起面对。
